# alz_speech_analysis

> A Python project to analyze language patterns for the early detection of Alzheimer's disease.

Questo progetto utilizza Python per analizzare marcatori linguistici nel parlato e nel testo per identificare indicatori precoci associati al declino cognitivo e alla malattia di Alzheimer.

## üöÄ Setup & Installazione

Questo progetto utilizza un ambiente virtuale Python.

1.  **Clona il repository:**
    - git clone [https://github.com/TUO_NOME_UTENTE/alz_speech_analysis.git](https://github.com/TUO_NOME_UTENTE/alz_speech_analysis.git)

    - cd alz_speech_analysis
    

2.  **Crea e attiva l'ambiente virtuale:**
    - python3 -m venv .
    
    - source .venv/bin/activate
    

3.  **Installa le dipendenze:**
    (Aggiungeremo un file `requirements.txt` qui non appena installi delle librerie)

    - pip install -r requirements.txt
    
## üèÉ Come Eseguire

Assicurati che il tuo ambiente virtuale sia attivo (`source .venv/bin/activate`) ed esegui lo script principale:

- python app.py
